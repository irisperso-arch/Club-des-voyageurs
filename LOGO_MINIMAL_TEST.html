4 versions — sans courbe complexe

Retour à l'essentiel. Aucune courbe de Bézier compliquée — seulement des formes qu'un code peut rendre proprement.

A — L'arc seul

Juste le cercle ouvert. L'ouverture dit le départ sans avoir besoin d'un trait supplémentaire. Très épuré.

B — Arc + point

L'ouverture est marquée par un seul point — la destination, le cap. Sobre et précis.

C — Arc + trait

Un trait droit part du bout de l'arc — pas de courbe, donc pas de problème. Direct comme une route.

D — Un seul geste

Un seul trait continu — entre dans le cercle, fait la boucle du club, en repart. Comme le symbole VW : une seule forme qui dit tout.

— A, B, C ou D ?

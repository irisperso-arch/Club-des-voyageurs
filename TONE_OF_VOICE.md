## 2. PERSONALITY

### Les 5 mots
**Décontracté • Authentique • Direct • Chaleureux • Drôle**

### Qui tu es
Tu es le pote qui :
- Revient de voyage et a des histoires à raconter
- Écoute les autres sans juger
- Rigole de ses propres gaffes
- Pose les vraies questions
- N'a pas peur de dire "ouais, ça c'était pas top"
- Donne des conseils pratiques sans te la péter

### Qui tu es PAS
- Le "je suis tellement cultivé" mec
- Le influenceur qui fait des poses
- Le corporate speak artist
- Le gars qui parle que pour s'écouter
- Le bullshit artist

## 3. COMMENT PARLER

### ✅ DO's - Ce qu'il faut faire

**Soyez direct & concis**
```
✓ "T'es allé en Islande ? C'est dingue."
✗ "L'Islande offre une expérience transformationnelle..."
```

**Utilisez la langue naturelle**
```
✓ "Ça craint d'être perdu dans un village de 200 personnes."
✗ "L'égarement dans des contextes ruraux pose des défis existentiels."
```

**Parlez vrai**
```
✓ "Le voyage, c'est cool. Mais aussi un peu dur. Et pas mal d'argent."
✗ "Chaque voyage est une aventure précieuse qui enrichit l'âme."
```

**Soyez curieux**
```
✓ "Attends, tu dis quoi ? T'as eu peur ?"
✗ "Veuillez détailler votre expérience émotionnelle."
```

**Utilisez "on"**
```
✓ "Nous, voyageurs belges, on aime bien..."
✓ "Qu'est-ce qu'on en pense ?"
✗ "Vous, voyageur, devriez..."
```

**Soyez spécifique**
```
✓ "À Marrakech, dans la médina près de la place Jemaa El-Fnaa..."
✓ "J'ai mangé des tajines chez cette mère qui parlait pas anglais."
✗ "Des lieux exotiques offrent des expériences culinaires authentiques."
```

**Rigolez de vous-même**
```
✓ "J'ai bouffé de la nourriture bizarre. Zéro regrets. OK un peu."
✓ "J'ai pris la mauvaise route. Trois fois."
✗ "Les erreurs de navigation enrichissent le parcours du voyageur."
```

### ✗ DON'Ts - Ce qu'il faut éviter

**Ne soyez pas prétentieux**
```
✗ "Les voyageurs cultivés comme nous..."
✗ "Ceux qui ne comprennent pas..."
✓ "Nous, on a trouvé que..."
```

**N'utilisez pas de jargon marketing**
```
✗ "Experience transformationnelle" / "Journey authentique" / "Mindfulness voyage"
✗ "Synergies culturelles" / "Écosystème de voyage"
✓ "Cool à voir" / "Ça change ta perspective" / "T'apprends des trucs"
```

**Ne forcez pas les hashtags**
```
✗ #wanderlust #instatravel #explorersociety #travelgram
✓ #ClubDesVoyageurs #PartirRevenirRaconter
```

**N'utilisez pas trop d'emojis**
```
✗ "C'était 🔥🌍✈️🏖️🙏🎉"
✓ "C'était fou. 🌍"
```

**N'exagérez pas**
```
✗ "C'était THE most amazing life-changing experience EVER!!!"
✓ "C'était vraiment bien. Ça m'a fait réfléchir sur un truc."
```

**Ne faites pas l'influenceur**
```
✗ "Swipe up to discover my fave Bali spots!"
✗ Selfies partout, poses, filters lourds
✓ Vraies photos, vraies moments
```

```
Pas la BS marketing. Les vraies questions.

Dans cet épisode :
• Pourquoi on revient différent
• Ce que les guides touristiques ne te disent pas
• Comment gérer le retour à la réalité

🔔 S'abonner pour les prochains épisodes

Partir, revenir, raconter.
```

## 5. KEY MESSAGES

### Message 1: Direct & Real
**"Des conversations vraies, pas du contenu"**

*Use when*: Explaining what makes the show different

```
"Contrairement aux autres émissions, on n'essaie pas de te vendre un rêve.
On parle de ce qui se passe vraiment quand tu voyages.
Les trucs cool. Les trucs qui craintent. Tout."
```

### Message 2: Expert But Chill
**"L'expertise sans la BS"**

*Use when*: Positioning the show

```
"Nos invités savent vraiment de quoi ils parlent.
Mais ils parlent pas comme des profs. Comme des potes."
```

### Message 3: For Real Travelers
**"Pour ceux qui voyagent pour vrai"**

*Use when*: Targeting audience

```
"Si tu voyages pour comprendre, pour rencontrer, pour te challenger —
c'est pour toi.

Si tu voyages pour les photos Instagram — euh, peut-être pas."
```

### Message 4: Authentic Stories
**"Les vraies histoires. Les gaffes incluses."**

*Use when*: Building credibility

```
"On parle des moments bizarres, des erreurs, des coups durs.
Pas juste les sunset photos."
```

### Message 5: Always Relevant
**"T'as un voyage en tête? On peut t'aider à comprendre ce qui t'attend."**

*Use when*: CTA / encouraging subscriptions

```
"Chaque épisode te donne des vraies insights.
Pas des listicles bullshit.
Des histoires qui te font réfléchir."
```

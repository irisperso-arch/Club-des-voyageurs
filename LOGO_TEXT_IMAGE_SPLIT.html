Le texte porte "club", le symbole porte "horizon"

Le nom "Le Club des Voyageurs" dit déjà tout : "Club" = talk-show/conversation, "Voyageurs" = désir de partir. Le symbole est donc libéré de devoir illustrer le club — il peut se concentrer à 100% sur l'envie d'évasion, sans redondance.

1. Le symbole seul — "horizon pur", épuré

Lever d'horizon

Un horizon, un soleil qui se lève (ou se couche — l'ambiguïté est volontaire : "partir, revenir"), une trajectoire en pointillés qui y mène. Rien d'autre. Lisible en 1 seconde, même en 48px.

Variante "route"

Même logique, traitement plus "chemin de vie" : un point de départ ancré (bleu marine), une route qui monte vers une destination ronde (corail). Plus dynamique, bonne pour les animations d'intro.

2. En situation — le symbole + le nom complet

Lock-up horizontal

Le mot "Club" est là, en toutes lettres, en Playfair Display — il porte l'identité talk-show. Le symbole, lui, ne montre que l'horizon. Les deux se complètent sans se répéter : aucune redondance visuelle.

Avatar / favicon (symbole seul)

Là où le nom disparaît (avatar de chaîne YouTube, favicon, app), le symbole tient seul et reste immédiatement reconnaissable comme "voyage / horizon / évasion" — exactement ce qui donne envie de cliquer, sans avoir besoin du mot "club" pour fonctionner.

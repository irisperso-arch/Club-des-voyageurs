Le C — marque principale

Le "C" de Club comme signature visuelle forte, à la façon dont C dans l'air utilise son C — reconnaissable seul, avant même le nom complet.

Sur fond sombre

Lock-up complet

B — C dans l'ovale (fusion badge + lettre)

C dans l'ovale sauge
